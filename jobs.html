Network Administrator (REF: 00001)

A Network Administrator is responsible for maintaining computer networks and ensuring their reliability and security.

Salary Range: $75,000 - $115,000 per year

Reports To: IT Manager

Key Responsibilities

  • Manage and maintain network infrastructure
  • Monitor network performance and troubleshoot issues
  • Ensure network security and compliance with policies
  • Install and configure network hardware and software
  • Provide technical support for network-related issues

Required Qualifications, Skills, and Attributes

Essential

  • Bachelor’s degree in Information Technology, Computer Science, or related field
  • 3 years of experience in network administration
  • Proficiency in configuring routers and switches
  • Strong troubleshooting and problem-solving skills

Preferable

  • Certifications such as CCNA
  • Experience with cloud networking

Software Developer (REF: 00002)

A Software Developer is responsible for designing, coding, and maintaining applications.

Salary Range: $80,000 - $120,000 per year

Reports To: Lead Software Engineer

Key Responsibilities

  • Develop and maintain software applications
  • Collaborate with teams
  • Write clean and efficient code
  • Participate in code reviews
  • Debug and troubleshoot software issues

Required Qualifications, Skills, and Attributes

Essential

  • Bachelor’s degree in Computer Science or related field
  • 3+ years of experience in software development
  • Proficiency in Python, or C++
  • Strong problem-solving skills

Preferable

  • Experience with cloud technologies
  • Familiarity with Agile frameworks
